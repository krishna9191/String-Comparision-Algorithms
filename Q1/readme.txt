

Execution: javac Q1Driver.java
			java Q1Driver
			
The Driver program takes four parameters:
			  args[0] - Test String
	  		  args[1] - pattern
	  		  args[2] - Algorithm selection parameter(0-5)
	  		  args[3] - y/n to print pattern occurrence
		

0 - Run all Algorithms \n1 - Naive \n2 - Rabin Karps \n3 - KMP \n4 - Boyer Moore-Bad Char\n5-Boyer Moore-Good Suffix

	  
The inputs can also be given from console.

SampleIO-1, SampleIO-2, SampleIO-3 contains test cases for Q1Driver.java
